## 🎨 Customization

### **Create Custom Gestures:**

```python
from Backend.JarvisGesture import jarvis_gesture

# Add custom gesture
jarvis_gesture.gesture_actions["wave"] = lambda: print("Hello!")

# Map to system action
jarvis_gesture.gesture_actions["point"] = lambda: pyautogui.click()
```

### **Adjust Sensitivity:**

```python
# Pinch sensitivity
jarvis_gesture.pinch_threshold = 30  # More sensitive

# Swipe threshold
jarvis_gesture.swipe_threshold = 80  # Easier swipes

# Smoothing
jarvis_gesture.smoothing = 5  # Smoother movement
```
