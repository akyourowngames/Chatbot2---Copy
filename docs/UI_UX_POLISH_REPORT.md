## 🎯 ANIMATION DETAILS

### **Timing Functions:**
```css
Primary:     cubic-bezier(0.16, 1, 0.3, 1)
Purpose:     Smooth, premium feel
Duration:    0.3s - 0.4s (perfect balance)
Delay:       Staggered (0.1s increments)
```

### **Key Animations:**

**1. Fade In Down (Header):**
```css
From: opacity 0, translateY(-30px)
To:   opacity 1, translateY(0)
Time: 0.8s
```

**2. Fade In Up (Cards):**
```css
From: opacity 0, translateY(30px)
To:   opacity 1, translateY(0)
Time: 0.8s
Delay: Staggered per card
```

**3. Message Slide In:**
```css
From: opacity 0, translateY(20px)
To:   opacity 1, translateY(0)
Time: 0.4s
```

**4. Pulse (Status):**
```css
0%:   scale(1), opacity 1
50%:  scale(1.5), opacity 0
100%: scale(1), opacity 1
Time: 2s infinite
```

**5. Shimmer (Progress):**
```css
From: translateX(-100%)
To:   translateX(100%)
Time: 1.5s - 2s infinite
```

**6. Typing Bounce:**
```css
0%:   translateY(0)
30%:  translateY(-10px)
60%:  translateY(0)
Time: 1.4s infinite
```
