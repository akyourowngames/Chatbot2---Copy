## 💡 Pro Tips

### **Customization:**
Edit `static/css/auth-modal.css`:
```css
:root {
    --auth-primary: #00d4ff;    /* Change colors */
    --auth-secondary: #7c3aed;
    --auth-accent: #f59e0b;
}
```

### **Add More Particles:**
In `auth-client.js`, add more:
```html
<div class="auth-particle"></div>
```

### **Change Animations:**
Modify keyframes in CSS:
```css
@keyframes logoFloat {
    /* Customize animation */
}
```
